Quick Start Guide

💡 Working with Keyframes

To animate, toggle onto Animation Mode, select any stroke, position the timeline playhead scrubber to any timing, change its location/scale using manual inputs, and hit Keyframe Property. Move the scrubber and repeat to generate linear interpolated translations!

🖼️ Backdrop Reference Control

Upload backdrop layouts in Ref Edit Mode. Rotate and translate your reference templates. When you switch back to Draw Mode, they automatically lock, ensuring uninterrupted drawings.

🎮 Use in Games

The structure is engineered to look like this in your game code loops:

// Simple Lerping inside update cycle:
function lerp(a, b, t) {
  return a + (b - a) * t;
}
// Get current value at 'time' between Keyframes
